# Online Game SET

## Prerequisites

### Install Dependencies
```
npm install
or
yarn install
```

### Install gulp
To build the client
```
npm install --global gulp
or
yarn global add gulp
```

## Build Client
```
gulp
```

## Run Server
```
node server.js
```
